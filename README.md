# Comparative DNA Sequencial Analysis Using BLAST

## Overview

This project utilizes the BLAST DNA database to compare the human TP53 gene with other homologous sequences from multiple vertebrate species to investigate the conservation of eukaryotic DNA and the evolutionary relationship between these relationships. 

## Objectives

* Download DNA sequences from known databases
* Perform BLAST sequential comparisons
* Analyze sequence similarity
* Visualize results
* Interpret biological significance between the genes

## Tools Used

* R
* BLAST
* FASTA
* NCBI database
